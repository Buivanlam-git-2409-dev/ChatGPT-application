# ChatGPT-application

Dự án này hướng dẫn cách sử dụng ChatGPT và các mô hình ngôn ngữ lớn thông qua thư viện OpenAI và LangChain, tích hợp với Groq Cloud.

## Cài đặt

Để chạy ứng dụng này, bạn cần cài đặt các thư viện cần thiết bằng cách chạy các lệnh sau:

1. **Cập nhật thư viện OpenAI:**
   ```bash
   pip install --upgrade openai
   ```

2. **Cài đặt các thư viện LangChain và Groq:**
   ```bash
   pip install langchain-classic langchain-groq langchain-core
   ```

3. **Cấu hình API Key:**
   Dự án sử dụng API Key từ [Groq Cloud](https://console.groq.com/) để truy cập các mô hình như Llama 3.3, Mixtral và Gemma.

## Các tính năng chính
- Phân loại văn bản (Text Classification)
- Dịch thuật (Machine Translation)
- Trả lời câu hỏi (Question Answering)
- Hội thoại đóng vai (Role Playing)
- Lập trình (Coding Applications)
